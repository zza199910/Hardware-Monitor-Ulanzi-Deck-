# Hardware Monitor — Ulanzi Deck Plugin

[![Available on Ulanzi Community Store](https://raw.githubusercontent.com/narlei/ulanzicommunitystore/main/docs/badges/ulanzi-community-store.svg)](https://ulanzicommunitystore.narlei.com)

A real-time hardware monitoring plugin for Ulanzi D200/D200H/D200X stream controllers. Displays CPU/GPU/RAM/Fan/Network metrics directly on your LCD keys with customizable visual styles.

> **Generated by DeepSeek-V4-Pro**

## Features

- **14 monitoring metrics**: CPU/GPU temperature, usage %, power, clock speed, GPU VRAM, RAM, fan speed, network upload/download
- **Real-time data**: Direct connection to LibreHardwareMonitor HTTP API (port 8085)
- **4 visual presets**: Gaming, Minimal, Retro, Cyberpunk — each with unique fonts, colors, and effects
- **2 chart styles**: Ring gauge or Wave chart
- **Settings persistence**: Remembers your configuration across restarts
- **Multi-language**: English, Chinese (Simplified/Traditional), German, Spanish, Japanese, Korean, Portuguese

## Requirements

- Ulanzi Studio v3.1.9
- [LibreHardwareMonitor](https://github.com/LibreHardwareMonitor/LibreHardwareMonitor) running with HTTP server enabled (port 8085)

## Installation

1. Copy the `com.ulanzi.HardwareMonitor.ulanziPlugin` folder to:
   ```
   %APPDATA%\Ulanzi\UlanziDeck\Plugins\
   ```
2. Restart Ulanzi Studio
3. Ensure LibreHardwareMonitor is running with its HTTP server on port 8085

## Usage

1. Drag any Hardware Monitor action to an LCD key
2. Select a **Preset Style** from the dropdown (Gaming / Minimal / Retro / Cyberpunk)
3. Choose **Chart Type** (Ring or Wave)
4. The key will display real-time hardware data

## Data Source

The plugin reads live hardware sensor data directly from LibreHardwareMonitor's HTTP API at `http://127.0.0.1:8085/data.json`. All parsing is done in pure JavaScript within the WebView — no external bridge service required.

If LHM is not running, the plugin gracefully shows zero values.

## Technical Details

- **Canvas**: 144×144 pixels, rendered via Canvas 2D API
- **Font**: Source Han Sans SC (provided by Ulanzi Studio)
- **Data refresh**: 1 second interval, global publish-subscribe pipeline
- **Settings persistence**: SDK `getSettings` / `didReceiveSettings` API
- **LHM parsing**: Custom tree-traversal parser for LibreHardwareMonitor's 93KB JSON sensor tree
